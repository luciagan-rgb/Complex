## 运行（推荐）

在 `3-4XXXX/` 目录启动一个静态服务器：

```bash
cd "/Users/a123/Desktop/课题/布局计算/3-4XXXX"
python3 -m http.server 8000
```

然后在浏览器打开 `http://localhost:8000`。
